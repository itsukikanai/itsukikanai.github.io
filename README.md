# itsukikanai.github.io

## ディレクトリ構造

```
/
│
├── index.html                         // ホームページ
│
├── stylesheets/                       // CSSファイルを格納するディレクトリ
│   ├── style.css                      // メインのスタイルシート
│   └── reset.css                      // リセット用のスタイルシート
│
├── scripts/                           // JavaScriptファイルを格納するディレクトリ
│   └── main.js                        // メインのJavaScriptファイル
│
├── assets/                            // アセットを格納するディレクトリ
│   ├─── images/                       // 画像を格納するディレクトリ
│   │    ├─── favicon/                 // ファビコンを格納するディレクトリ
│   │    │    ├── favicon.svg          // ファビコン画像（SVG）
│   │    │    ├── favicon.ico          // ファビコン画像（ICO）
│   │    │    └── apple-touch-icon.png // Webクリップアイコン
│   │    └─── logo.svg                 // ロゴ画像
│   ├── fonts/                         // フォントファイル
│   ├── audio/                         // 音声ファイル
│   ├── videos/                        // 動画ファイル
│   ├── documents/                     // 文書ファイル
│   ├── spreadsheets/                  // 表計算ファイル
│   ├── presentations/                 // プレゼンテーションファイル
│   └── models/                        // 3Dモデルファイル
│
├── error/                             // エラーページを格納するディレクトリ
│   └── 404/                           // 404エラー
│
└── blog/                              // ブログページを格納するディレクトリ
```
